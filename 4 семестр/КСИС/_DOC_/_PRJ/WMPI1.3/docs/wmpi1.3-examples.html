WMPI 1.3 Examples Documentation
    This is a listing of the examples that accompany the current release of WMPI. Visit WMPI for more examples or links to examples.
    The listing provides some insight into the examples and the functions used within each example.

Examples in C


 
Console_pi – Computes the value of Pi (to a degree chosen by the user) through the calculation of an integral. Each process computes the area under the graph between an interval and the coordinating process adds up the areas from each process. The output from this process is the value of the calculated Pi as well as statistics regarding the time spent on the computation and the communication.
MPI_Send, MPI_Receive, MPI_Barrier
Good for starters. This is a simple example that illustrates the tip of the MPI iceberg.
Gui_pi – same as console_pi except that, as the name implies, this version offers the potential of a GUI due to the project properties.
MPI_Send, MPI_Receive, MPI_Barrier
Basically, same as the console_pi example.
IO async – each process writes to separate files and reads them back; uses asynchronous IO.
MPI_Abort, MPI_Bcast, MPI_File_open, MPI_File_set_view, MPI_File_iwrite, MPIO_Wait, MPI_File_close
IO coll_perf – measures the IO bandwidth for a 3D block-distributed array to a file corresponding to the global array in row-major (C) order.
MPI_Abort, MPI_Bcast, MPI_Dims_create, MPI_Type_create_darray, MPI_Type_commit, MPI_Type_size, MPI_File_open, MPI_File_set_view, MPI_File_write_all, MPI_File_seek, MPI_File_read_all, MPI_File_close, MPI_Barrier, MPI_Wtime, MPI_Allreduce, MPI_Type_free
IO colltest – Uses collective I/O. Writes a 3D block-distributed array to a file corresponding to the global array in row-major (C) order, reads it back, and checks that the data read is correct.
MPI_Abort, MPI_Bcast, MPI_Dims_create, MPI_Type_create_darray, MPI_Type_commit, MPI_Type_size, MPI_Irecv, MPI_Send, MPI_Wait, MPI_File_open, MPI_File_set_view, MPI_File_write_all, MPI_File_seek, MPI_File_read_all, MPI_File_close, MPI_Type_free
IO cperf – a simple performance test.
MPI_Abort, MPI_Bcast, MPI_File_open, MPI_File_seek, MPI_Barrier, MPI_Wtime, MPI_File_write, MPI_File_seek, MPI_File_read, MPI_File_write, MPI_File_sync, MPI_File_close, MPI_Allreduce
IO csimple – each process writes to separate files and reads them back; console version.
MPI_Abort, MPI_Bcast, MPI_File_open, MPI_File_set_view, MPI_File_write, MPI_File_read, MPI_File_close
IO misc – tests various miscelaneous functions.
MPI_Abort, MPI_Bcast, MPI_File_open, MPI_File_write, MPI_File_sync, MPI_File_get_amode, MPI_File_get_atomicity, MPI_File_set_atomicity, MPI_Type_vector, MPI_File_set_view, MPI_File_get_view, MPI_File_get_byte_offset, MPI_File_get_type_extent, MPI_Type_extent, MPI_File_get_group, MPI_File_set_size, MPI_Barrier, MPI_File_get_size, MPI_File_seek, MPI_File_get_position, MPI_File_preallocate, MPI_File_close, MPI_File_delete
IO simple – each process writes to separate files and reads them back; GUI version.
MPI_Abort, MPI_Bcast, MPI_File_open, MPI_File_set_view, MPI_File_write, MPI_File_read, MPI_File_close

Examples in Fortran


 
Console_fpi – Computes the value of Pi (to a degree chosen by the user) through the calculation of the integral 4/(1 + x2). Each process computes the area under the graph between an interval and the coordinating process (node 0) adds up the areas from each process. The output from node 0 is the value of the calculated Pi as well as the approximate error of the calculation.
MPI_COMM_RANK, MPI_COMM_SIZE, MPI_BCAST, MPI_REDUCE
For starters. Simple example that shows the potential of collective operations under MPI. MPI_REDUCE is a function that permits many operations besides summing up numbers from all nodes in one node.
Gui_fpi – same as above, except that its project file is GUI
MPI_COMM_RANK, MPI_COMM_SIZE, MPI_BCAST, MPI_REDUCE
Matmult – Multiply a matrix times a vector, self-scheduling algorithm.
MPI_COMM_RANK, MPI_COMM_SIZE, MPI_ABORT, MPI_BCAST, MPI_SEND, MPI_RECV
ftp://info.mcs.anl.gov/pub/mpi/using/examples
Nbody – Calculates the force between N particles according to Newton's Law of Gravitation. This calculation is the kernel of particle simulation codes.
MPI_COMM_RANK, MPI_COMM_SIZE, MPI_BCAST, MPI_SEND, MPI_RECV, MPI_WTIME, MPI_SENDRECV_REPLACE, MPI_BARRIER, MPI_TYPE_VECTOR, MPI_TYPE_COMMIT
http://www.epm.ornl.gov/~walker/mpi/examples/nbody.html
Vectest – Example program to move data around a 3-d cube using a single processor and derived datatypes.
MPI_COMM_RANK, MPI_COMM_SIZE, MPI_BCAST, MPI_TYPE_VECTOR, MPI_WTIME, MPI_TYPE_HVECTOR, MPI_TYPE_COMMIT, MPI_SENDRECV, MPI_TYPE_FREE
